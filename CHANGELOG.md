## [0.1.0] - 2026-07-14

### Added

- Typed Python library built on SQLAlchemy 2.
- Safe table reflection and parameterized SQL exports.
- Chunked, atomic CSV, JSON, and JSON Lines writers.
- Optional Parquet and Feather support through PyArrow.
- Credential-safe command-line interface.
- Test suite, CI matrix, release workflow, and contributor documentation.

### Removed

- Legacy Python 2 Flask prototype and its unauthenticated arbitrary-query API.
